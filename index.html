I Enjoy

Chef
When it comes to infrastructure automation Chef is my favorite choice. The setup is pretty extensive especially if you go for the entire workstation / server / node setup. But then again what tool like this isn't a tough setup? Their Knife command is extremely easy to use to manage your cookbooks and nodes.

Flight
Extremely intuitive framework for developing RESTful based APIs. It's very light, very fast, and has very good routing techniques.

Codeigniter
When i'm not building an API centric web application I will use Codeigniter because of its small footprint and its ease of use and setup.

Amazon Web Services
Hands down the best cloud based service I have worked with. The reliability of AWS is great and they have all the tools necessary to get your web app up and running.

Digicert
Best SSL certificate provider I have found. They have amazing 24/7 support, great pricing, and their account area is very user friendly.

Bitbucket
Lots of my work requires private repositories and Bitbucket is great because they offer unlimited private repos.

Aptana Studio 3
My main preference for development. It has some good syntax highlighting and great integration with GIT. It's also makes it very convenient for me to have an entire section of servers that I can quickly connect to inside the application.
